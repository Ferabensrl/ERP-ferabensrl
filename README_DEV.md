# Feraben SRL PWA

Proyecto basado en Vite + React + TypeScript.

## Scripts

- `npm run dev` inicia el servidor de desarrollo.
- `npm run build` compila la aplicación.
- `npm run preview` sirve la versión compilada.
- `npm run import:productos` importa los productos desde el Excel provisto a la base de datos Supabase.

Variables de entorno requeridas para Supabase:

```
VITE_SUPABASE_URL=<url>
VITE_SUPABASE_ANON_KEY=<clave>
```
